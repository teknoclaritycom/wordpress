Tentang Tekno Clarity

Selamat datang di Layanan Tekno Clarity, Tekno Clarity adalah sebuah situs informatif yang berisikan konten seputaran dunia programming, bisnis, teknologi, sosial media, marketing dan berbagai pengetahuan umum serta hiburan menarik lainnya.


Tekno Clarity dibuat oleh Muh. Irvan Jaya di tahun 2022, Merupakan sebuah portofolio kemampuan pribadi sang pembuat dalam bidang web management, web development dan mobile apps development.


Adapun beberapa skil kemampuan yang dimiliki oleh pemilik adalah sebagai berikut:


- HTML

- CSS

- JavaScript

- Bootstrap

- Java

- Code Igniter

- Laravel

- Ionic

- Video Editing

- Search Engine Optimization

- Digital Marketing

- Content Writing


Beberapa kali jasa telah digunakan oleh berbagai pihak, contohnya pembuatan website PPID Universitas Mulawarman dan menjadi admin media sosial di masa PKL Kantor Bupati Kutai Kartanegara - Divisi Protokol dan Komunikasi Pimpinan.
